用語解説

用語 略称、英訳 解説
防御率ぼうぎょりつ ERA(Earned Run Average) 投手が9イニングあたり平均して何点を失うかを示す指標。「自責点×9÷投球イニング」で計算する。
自責点じせきてん ER(Earned Run) 投手の責任としてカウントされる失点の事。野手のエラーなどで「取れるアウトが取れず、そのアウトが取れていれば失点はなかった」ようなシチュエーションの失点が対象外となる。防御率の計算にはこの自責点が使われる。
ホールド H(Hold) ・先発でなく、自身に勝利、敗戦、セーブが記録されない
・1つ以上のアウトを取っている
・自分が試合最後の自軍ピッチャーでないこと
・交代後、自身に記録される失点でチームが同点または逆転されないこと
この条件下で「セーブと同じ条件下で投げる」または
同点の状況で登板し、同点のまま失点をせずに降板する
上記の条件を満たすと記録される。
ホールドポイント HP(Hold Point) 「ホールド+勝ち星」で計算されるもの。NPBにおいて、最優秀中継ぎ投手賞はこのHPが最も多い選手へ送られる。
セーブ S(Save) ・先発でなく、自身に勝利、敗戦が記録されない
・1つ以上のアウトを取っている
・自分が試合最後の自軍ピッチャーであること
上記の条件を満たしたうえで、
・登板時3点差以内で1イニング以上投げる
・登板した最初の打者と次の打者にHRを打たれたら同点or逆転の場面の登板
・点差は関係なく、3イニング以上投げる
上記の条件を満たすと記録される。
NPBにおいて、最多セーブ投手賞はこのセーブ数が最も多い選手へ送られる。
ワイルドピッチ WP(Wild Pitch) いわゆる「暴投」。ピッチャーの球が大きく逸れてキャッチャーが取れず、その結果ランナーが進塁してしまったときに記録される。
パスボール passed ball 暴投にあたらない、キャッチャーが取れるであろう球を後ろや横に逸らすなどしてランナーが進塁したときに記録される。パスボールとワイルドピッチの匙加減は記録員によって決められる。
牽制けんせい pickoff ピッチャーが投球前にボールをランナーのいるベースへ向かって投げる動作を指す。ランナーがベースに戻り切れずに先にベース上にいる野手にタッチされるとアウトとなる。
ボーク balk ピッチャーが牽制を行うときや投球する際にしてはいけない行為のこと。「投球動作を途中で止める」や「セットポジションで静止せずに投げる」などが該当する。
セットポジション set position ピッチャーが軸足(右投手なら右足、左なら左足)をプレートに付けて静止してから投げる姿勢の事。これに対して体がバッターに対して正面を向く姿勢から投球することを「ワインドアップ」という。
QS Quality Start 先発投手が6イニング以上を投げ、自責点が3点以内の試合。QSを記録すると、その試合は「クオリティスタート」と呼ばれる。
WHIP Walks plus Hits per Inning Pitched 投手が1イニングあたりどれだけランナーを出しているかを示す。
LOB% Left On Base Percentage 出塁させたランナーを生還させなかった割合の事。
FIP Fielding Independent Pitching 守備の影響を受けない投手の能力を示す指標。四球、三振、本塁打で算出する。投手のみの成績を計るために使われたりする。
RA Run Average 9イニング当たりの失点頻度。
RA- Run Average Minus RAをリーグ平均値と対比させたもの。平均100とし、低いほど優秀。50だとすると失点が平均的な投手と比較し50%に抑えられていることを表している。
tRA True Run Average FIPの項目に加え、どのような打球が打たれたかも含めて計算する指標。
R-F RA-FIP RAとFIPの差。投手が実力通りの結果を残しているかどうかの指標。
DER Defensive Efficiency Ratio ホームランを除いたグラウンド上に飛んだ打球のうちどれくらい野手がアウトにしたかを表す指標。投手ではなくチーム全体の守備力を表す指標。
用語 略称、英訳 解説
打率だりつ Avg,Av(Average)など 「安打数÷打数」で算出されるもの。一般的に3割を超えると優秀な打者とされやすい。
出塁率しゅつるいりつ OBP(On-base percentage) 「(安打+四球+死球)÷(打数+四球+死球+犠飛)」で計算されるもの。「打者がその打席でアウトにならない確率」という解釈が良い。
塁打るいだ TB(Total Bases) 単打、二塁打、三塁打、本塁打それぞれ1,2,3,4として計算され、その合計値の事。
長打率ちょうだりつ SLG(Slugging percentage) 「塁打÷打数」で計算されるもの。1打数あたりで平均どのくらいの塁打を稼いだかを表す。
得点とくてん R(Run) その選手がホームベースを踏み、チームに得点が与えられる時に増えるもの。
打点だてん RBI(Run batted in) その選手が打撃によって自チームへ得点をもたらしたときに増えるもの。安打,四死球,犠打が該当する。例外として、併殺打(ゲッツー)間に記録された得点では打点に加算されない。
盗塁とうるい SB(Stolen Base) 投手がバッターに投げた際に先の塁を狙い走ること。セーフになれば盗塁が記録され、アウトになると盗塁死が記録される。
犠打ぎだ,送りバントおくりばんと SH(Sacrifice hit) 塁にランナーがいるときに自身がアウトになってランナーを進める打撃の事。
犠飛ぎひ,犠牲フライぎせいふらい SF(Sacrifice Fly) 犠打とは別でカウントされる。フライ,もしくはライナーを相手の外野手が捕球した後、ランナーがホームに帰り得点が入ったときに記録されるもの。そのため、「フライ、ライナー捕球後に2塁から3塁への進塁」などは犠飛として記録されない。
併殺打へいさつだ GIDP(Grounded Into Double Play) 一つのプレーでフォースアウトにより2つのアウトを取ること。「ゲッツー」「ダブルプレー」とも呼ばれる。
スリーバント bunt with two strike 2ストライク後に行うバントのこと。このバントがファールボールになるとバッターはアウトになり、三振が記録される。
タイムリーヒット clutch hit,RBI hit 得点を伴うヒットのこと。1点ならタイムリーヒット、2点か3点だと得点に応じた「〇点タイムリー」などと表現される。
OPS On-base Plus Slugging 出塁率+長打率。打席あたりでの貢献値を表し、近年では打率よりも重要視されつつある。.700~.750あたりが平均を上回り、.800以上が優秀、.9を超えてくるとリーグを代表する打者レベル。
K% 打席のうち三振が占める割合
ISO 「長打率-打率」で算出する。バッターの長打力を表す。
BB% 打席のうち四球が占める割合
BABIP Batting Average on Balls In Play 「(安打-本塁打)÷(打数-三振-本塁打+犠飛)」で算出する数値。HR、ファールを除いた打球のうち、安打となった割合を表す。どの選手も試行回数を重ねるとリーグの平均値に収束していくとされている。
Spd 足の速さを表す指標。平均は5前後、盗塁成功率、盗塁の企画などをポイント化して評価する。
WAR Wins Above Replacement 「代替選手に比べてどれだけの勝利をもたらすか」を示す指標。打撃、走塁、守備、リーグ平均の選手と比較して算出される。
UBR Ultimate Base Running 盗塁と盗塁死を除いた走塁貢献を得点化した指標。平均的な走者と比較しどれほど走塁で得点を生み出したかを表す。
wRC+ Weighted Runs Created Plus 「リーグ平均を100としたときの得点生産力」を示す指標。150だったら平均的な打者と比較して1.5倍の貢献をしている、など。
wRAA Weighted Runs Above Average 「リーグ平均を0としたときの得点生産力」を示す指標。プラスなら平均以上、マイナスなら平均以下の得点生産力を持っている。
wSB Weighted Stolen Base runs 盗塁の貢献を得点化した指標。平均的な走者に比べどれだけ盗塁で多く得点を生み出したかを表す。wSB2であれば理論上は平均的な走者に比べてチームの得点が2増える、というもの。
wOBA Weighted On-Base Average バッターが打席あたりどれだけチームの得点を増やすのに貢献しているかを表す指標。平均はおよそ.330あたりとなる。
用語 略称、英訳 解説
フォースプレー Force play 前の塁に自分以外のランナーがいる際、自身は次の塁を狙わなければならない。そのような強制的に進塁が必要な場合にて、進塁先の塁を守備側がボールを持った状態で踏むことで自動的にランナーがアウトになるプレーのこと。
クロスプレー bang-bang play,close play アウトかセーフかを判断しづらいプレー。一般的には野手とランナーが接触しそうになるプレーを指す。
エンタイトルツーベース Ground rule double フェアゾーンへ転がった打球がバウンドするなどして観客席に入った場合に宣告されるもの。打者、ランナーはそれぞれ2つ進塁することができる。
サヨナラ Walk-Off 後攻チームが最終回に勝ち越すケースを指す。勝ち越した時点でそのチームの勝利は決まっており、その後攻撃する必要がない。そのためサヨナラゲームとなり、ゲームセットとなる。
フィルダースチョイス fielder's choice 「野選」と日本語では表記される。ランナーがいる状態で放たれた打球を処理するときにランナーをアウトにしようとしてアウトにできず、結果として1つもアウトを取れなかったときなどに記録される。
インフィールドフライ infield fly ノーアウト、もしくは1アウトでランナーが1,2塁または満塁の時にバッターが内野フライを打ち上げたときに宣告されるルール。宣告された時点でバッターはアウトとなる。わざと落球してダブルプレーを取られることを防ぐためにある。
タッチアップ tag up バッターがフライを打ち上げたとき、ランナーが帰塁(元いた塁へ戻る)し、野手がそのフライに触れた瞬間から次の塁へ進むことができること。
コリジョンルール collision rule かつてはランナーが本塁を狙うとき、アウトのタイミングでもセーフになるためにキャッチャーへタックルするなどして落球をさせようとする過激なプレーがあった。それによる負傷を避けるために制定されたルールである。 また、セーフのタイミングであってもボールを持っていないキャッチャーがホームベースを完全にふさぐ行為(ブロックという)によってアウトにされてしまうケースをなくす目的でもある。
スコアリングポジション scoring position 「得点圏」と呼ばれ、ランナーが2塁、または3塁にいることを表す。ヒット1本でランナーが本塁へ帰ってこれる可能性が高いことからこう呼ばれる。
ノーヒットノーラン No-hitter 投手がヒットを許さず、完全試合を達成しなくても良いが、ノーヒットで試合を終えたことを指す。完全試合とは異なり、四球やエラーでランナーを出しても良い。
完全試合 Perfect Game 投手がヒット、四球、エラーなどを一切許さず、完全に無安打無四球で試合を終えたことを指す。
サイクルヒット Cycle 1試合で単打、二塁打、三塁打、本塁打を記録したことを指す。
